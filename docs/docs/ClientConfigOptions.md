# ClientConfigOptions

Config options for a client.

## Example
```json
logCategory: "851163590811320343",
color: "#7289da"
```
## Types
- Object

## Properties

PARAMETER | TYPE | OPTIONAL | DEFAULT | DESCRIPTION
------ | ------ | ------ | ------ | ------
color | [Discord.ColorResolvable](https://discord.js.org/#/docs/main/stable/typedef/ColorResolvable) | true | `DEFAULT` | The color used to send system messages.
logCategory | String | true | None | The log category used to send system messages.
