## About
Discord.jsh is a package built to make making a bot easier!

- [Examples](/examples)
- [Starting](#Installation)

## Installation
*Node.js 16.6.0 or newer is required.*

```shell
# NPM
npm i discord.jsh
# Yarn
yarn add discord.jsh
```
